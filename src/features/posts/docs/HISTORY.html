Posts History

Implementation decisions and later changes.

Initial Boards & Post Implementation

2026-06-27

Added the first post creation slice with location-aware board resolution, Effect Schema validation, SQLite persistence, local evidence uploads, a safe evidence reader, and a floating creation pane over the map.

Location Model

The task clarified that location is not one enum. It is a hierarchy: required country, optional state or union territory, and optional district. The database therefore stores country, state, and district separately.

Evidence Storage

Local upload support was selected for v1. Uploaded files are stored under git-ignored data/uploads/post-evidence with generated filenames, while the post row stores only the evidence URI.

Board Source Data

The board constants include source metadata and status. Union ministries are represented as verified country-level boards. Every state and union territory receives a state-level department board set so the UI and validation paths work consistently, while jurisdictions that still need official department-by-department verification remain marked as incomplete.

Evidence Extension Normalization

Uploaded evidence filenames now use the extension implied by the validated content type instead of preserving the user's original filename extension. This keeps generated evidence URIs readable by the evidence-serving route even when the uploaded filename has a misleading extension.

Timestamp Control Refinement

Replaced the combined timestamp picker with side-by-side date and time fields. The date field uses a shadcn calendar popover, and the time input initializes to the user's current local time before a date is selected.

File Picker Affordance Refinement

2026-06-28

Reworked the file evidence input so users see a distinct choose-file segment, a divider, and a filename display segment. Both visible segments point at the same native file input, preserving accessible file selection and form submission behavior.

Mobile Pane Overlay Refinement

2026-06-28

Adjusted the sheet overlay layering used by the mobile post creation pane. The backdrop can still dim and blur the map canvas, but the sidebar content is forced above the overlay so the form remains interactive on smaller screens.

Pane Close Reset

2026-06-28

Added a close-state reset for the post creation pane so native form fields, controlled selectors, uploaded filename display, timestamp state, submit state, and status messages return to their defaults whenever the sidebar closes.

File Picker Label Behavior

2026-06-28

Aligned the file evidence picker with the custom select controls: the field label names the control but does not trigger it. Only the visible choose-file and filename display control opens the native file picker.
